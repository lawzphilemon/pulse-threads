# Mekanik Threads 2026 — grounding

*Snapshot Juli 2026. Aturan kedaluwarsa: kalau environment punya web search dan file ini sudah berumur > 3 bulan → verifikasi ulang ranking signal Threads sebelum sesi strategi pertama, laporkan perubahan, sarankan update file ini.*

## Fakta algoritma

- Distribusi bertahap: post baru dites dulu ke sebagian kecil audiens (± 2–5% follower). Performa 30–60 menit pertama menentukan apakah post didorong lebih luas atau berhenti.
- Sinyal ranking terkuat: **reply substantif** — percakapan bolak-balik bernilai lebih dari likes. Reply murahan ("nice post", emoji-only) dan engagement bait di-downrank.
- Baris pertama = segalanya. Feed memotong ± baris pertama dengan tombol "more"; rasio tap-through vs scroll diukur sebagai sinyal kualitas.
- Rebalance 2025: konten dari akun yang sudah di-follow dinaikkan porsinya → segelintir *regulars* yang rutin reply adalah mesin distribusi jangka panjang, bukan sekadar viral one-off.
- Link: penalti lama sudah dibalik — Mosseri konfirmasi (2025) ranking kini memberi nilai lebih pada post ber-URL. Tapi sebagian praktisi masih mengamati performa lebih lemah → perlakukan link sebagai **variabel eksperimen**, bukan asumsi aman. Default: link di reply pertama, bukan di post utama, sampai data akun ini bilang sebaliknya.
- Konten original diprioritaskan; cross-post verbatim dari X/LinkedIn cenderung mati — tone X (broadcast, combative) tidak transfer ke Threads (curious, percakapan).
- Sinyal Instagram memengaruhi rekomendasi Threads (termasuk profile view di IG) → sinergi langsung dengan carousel IG Lawrence: tema Threads dan carousel bisa saling memanaskan.
- Fitur relevan 2026: Topic Tag (pakai 1 per post, relevan pilar) · Communities & Communities Hub · Dear Algo / Your Algo · Account Status (cek kelayakan post direkomendasikan ke non-follower) · DM · ghost post.
- Cadence sehat: 2–5 post/minggu + reply bermakna di post orang lain hampir tiap hari.

## Konteks Indonesia

- Indonesia ± 12,2 juta pengguna Threads, peringkat 4 dunia (April 2026); momentum platform naik, DAU global Threads melewati X sejak awal 2026.
- Kultur Threads Indonesia: curhat/confessional, spill, cerita personal dengan angka nyata, keresahan finansial harian — persis wilayah konten Lawrence (biaya RS, sandwich generation, orang tua menua, gaji vs proteksi).
- **Tes wajib sebelum setiap post dianggap lolos:** kalau post terasa seperti *broadcast* → gagal. Kalau terasa seperti *kalimat pertama sebuah percakapan* → jalan.
